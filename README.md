# transitdash
A dashboard system for Waveshare 5.83" e-ink display with React frontend and node.js backend

requires:
metra api key - https://metra.com/metra-gtfs-api
cta train api key - https://www.transitchicago.com/developers/traintrackerapply/
cta bus api key - https://www.transitchicago.com/developers/bustracker/
pace only offers gtfs data - https://www.pacebus.com/route-timetable-data-services
